# Parrot App

Demo app  %100 kotlin

Parrot app that loads information from Parrot API to show one approach to using some of the best practices in Android Development. 

Including:

- ViewModel
- LiveData
- Hilt (for dependency injection)
- Kotlin Coroutines
- Retrofit
- Room
- WorkerManager
- Mockk for Coroutines
- Testing with JUnit4
